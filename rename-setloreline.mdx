Команды `/rename`, `/setloreline` и [другие](/o/DW2qGgvIqDS2hOdKc27R/s/n9tgebG4jFUoqtzRTUxk/igrovoi-kontent/rename-setloreline#commands) - это платная услуга входящая в подписку [☀Солнышко](https://luminous.easydonate.ru/), позволяющая вам редактировать название и описание любого предмета.

### **Доступные цвета**

Мод поддерживает [ванильные цвета](https://app.gitbook.com/o/DW2qGgvIqDS2hOdKc27R/s/n9tgebG4jFUoqtzRTUxk/igrovoi-kontent/rename-setloreline#native-colors) Minecraft и HEX цвета (`#RRGGBB`).

Для использования HEX, необходимо поставить `&` перед цветом. Например: `&#RRGGBB`

Для отключения курсива используйте `&o`, для жирного текста `&l` 

​[Удобный сайт для генерации градиентов](https://www.birdflop.com/resources/rgb/)​

#### **Ванильные цвета**

| **Код** | **Название** | **Цвет** |
| :-- | --: | :-: |
| `&0` | Чёрный | #000000 |
| `&1` | Тёмно-синий | #0000AA |
| `&2` | Тёмно-зелёный | #00AA00 |
| `&3` | Темно-бирюзовый | #00AAAA |
| `&4` | Тёмно-красный | #AA0000 |
| `&5` | Тёмно-фиолетовый | #AA00AA |
| `&6` | Золотой | #FFAA00 |
| `&7` | Серый | #AAAAAA |
| `&8` | Тёмно-серый | #555555 |
| `&9` | Синий | #5555FF |
| `&a` | Зелёный | #55FF55 |
| `&b` | Бирюзовый | #55FFFF |
| `&c` | Красный | #FF5555 |
| `&d` | Светло-фиолетовый | #FF55FF |
| `&e` | Жёлтый | #FFFF55 |
| `&f` | Белый | #FFFFFF |

#### **Команды**

| **Команда** | **Назначение** |
| :-- | :-- |
| `/rename <новое название предмета>` | Редактирование предмета |
| `/addloreline <текст>` | Добавление новой строки в конец лора |
| `/setloreline <номер строки> <текст>` | Изменение или добавление строки лора с конкретным обозначением |
| `/removeloreline <номер строки>` | Удаление строки лора |
| `/removelore` | Полная очистка лора |
| `/removename` | Вернуть предмету изначальное имя |
| `/lockitem` | Заблокировать изменение предмета |
| `/unlockitem` | Разблокировать изменение предмета |
